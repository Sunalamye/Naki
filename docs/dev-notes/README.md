## 筆記格式規範

每個開發筆記應包含：

1. **背景** - 為什麼需要這個功能
2. **探索過程** - 如何找到實現方法
3. **實現方案** - 最終的技術方案
4. **代碼位置** - 相關的檔案和行號
5. **測試結果** - 驗證功能是否正常
